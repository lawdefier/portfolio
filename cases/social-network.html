Social Network System (C++)

A low-level C++ social network modeling user relationships, post feeds, and friend suggestions from first principles.

C++ Developer
Modern C++ (C++17)
BST, Graph, Queue, Hash Table

Overview

Built completely without external high-level frameworks or Standard Template Library (STL) dependencies to master low-level memory allocation, algorithmic complexity, and graph processing fundamentals.

Data Structures Architecture

  • Graph (Adjacency List): Maps bidirectional user friendships and computes mutual friend counts via Breadth-First Search (BFS).
  • Binary Search Tree (BST): Handles fast user profile lookup by username with O(log N) average time complexity.
  • Queue Feed Buffer: Maintains chronological timeline posts with dynamic FIFO buffering.
