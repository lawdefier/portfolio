Student CGPA Predictive Analytics

Predicting academic outcomes from student behavioral habits using Ridge, Lasso, and XGBoost models coupled with SHAP explainability.

Data Science Lead
Python, Scikit-Learn, XGBoost, SHAP
~90% R² Score
Behavioral & Study Habit Survey Data

Problem Statement

Identifying early indicators of student academic struggle allows universities to offer timely academic support. However, raw academic scores rarely explain *why* performance drops. This project built a predictive pipeline linking study patterns, sleep routines, and attendance habits directly to predicted CGPA metrics.

ML Modeling & Pipeline

We engineered features from survey responses, performed outlier removal via IQR filtering, and evaluated multiple regression algorithms:

  • Ridge & Lasso Regression: Established baseline linear relationships and evaluated L1/L2 regularization penalties to prevent overfitting.
  • XGBoost Regressor: Captured non-linear interactions between study hours, sleep schedules, and course load. Achieved peak R² accuracy near ~90%.
  • SHAP (SHapley Additive exPlanations): Provided model explainability, highlighting daily study consistency and attendance as the top drivers of high CGPA.
# Model Evaluation Pipeline (Python excerpt) from xgboost import XGBRegressor from sklearn.metrics import mean_squared_error, r2_score import shap xgb_model = XGBRegressor(n_estimators=100, max_depth=4, learning_rate=0.05) xgb_model.fit(X_train, y_train) y_pred = xgb_model.predict(X_test) print(f"R2 Score: {r2_score(y_test, y_pred):.4f}") print(f"RMSE: {mean_squared_error(y_test, y_pred, squared=False):.4f}") # Compute SHAP feature contributions explainer = shap.Explainer(xgb_model) shap_values = explainer(X_test)

Key Takeaways

Combining tree-based gradient boosting with SHAP values transformed a black-box model into actionable educational insights, proving that study routine consistency outweighs cramming hours.
